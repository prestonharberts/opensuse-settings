# Discord

Run the following to install Discord:

```
flatpak --user install flathub com.discordapp.Discord -y
```

Change the following settings:

- Notifications
  - Badges
    - Enable Unread Message Badge: Off
- Appearance
  - Theme: Dark
  - UI Density: Spacious
- Linux Settings
  - System Startup Behaviour
    - Open Discord: Off
  - Close Button
    - Minimize to Tray: Off
